Hexo Blog 搭建笔记

安装

关于安装，可以参考的Hexo权威文档是谁呢？只有一个：Hexo 官方文档。记得前几年还是旧版 Hexo 2 时候，好些功能实现还是非常受限的，比如默认不支持本地端口实时预览等等。所以，如果想参考百度搜索下来的 Hexo 博客建站教程，请要多提个心眼！

安装过程请完全参考 Hexo 官方文档，我是 Mac 用户，所以我的安装步骤是（安装前提 Node.js 和 Git）：(Mac 用户要有安装过Xcode这个鬼东西)

xxxxxxxxxx

1
$ npm install -g hexo-cli

建站

建站的过程简单到爆：创建一个文件夹储存自己的 blog，它会自动 clone 一些文件目录到这个文件夹中；然后 cd 到这个文件夹目录下；用 npm 安装 package.json 中记录的各种版本程序包。

xxxxxxxxxx
3
1
$ hexo init <folder>
2
$ cd <folder>
3
$ npm install

Markdown 和 HTML 文件会被解析并放到 public 文件夹（若没有，则会新建），而其他文件会被拷贝过去。

操作到这里后，我们应该简单的测试一下，以确保步骤进行到这里一切都是正常的，分别依次键入：

1

2
$ hexo s        # 启动服务器

此时，一个默认 theme 的博客站点在 http://localhost:4000 这个本地端口中可以用浏览器预览到。
