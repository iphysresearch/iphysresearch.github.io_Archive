# Guest Lecture. Efficient Methods and Hardware for Deep Learning



## The Challenge

- Model Size
  - 对于超过 100 MB 的内容，不连 WiFi 是不会下载的。所以，对类似 Baidu 或 Facebook 这样的产品经理，他们对模型的二进制文件的大小非常敏感；自动驾驶的汽车智能在线更新模型。
- Speed
  - 训练速度很慢。
- Energy Efficiency
  - 能耗问题。嵌入式设备上会降低续航时间；在大型数据中心会增加运维成本。
