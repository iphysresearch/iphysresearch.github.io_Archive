# Artificial Neural Networks

**人工神经网络**

---

## Course Description

基本内容：

1. 大脑中神经细胞及其组成的生物神经网络。
2. 人工神经网络的特点，发展历史和基本类型。
3. 前馈型人工神经网络(感知器、BP网络)。
4. 反馈型人工神经网络(Hopfield、CNN网络)。
5. 竞争神经网络与自组织映射模型。 
6. 径向基函数网络(RBF网络)。
7. 深度学习理论与算法。
